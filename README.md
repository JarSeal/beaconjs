# Design Documenter

## Requirements

- node (>=16.13.2)
- yarn (>=1.22.15)
- docker
- docker-compose

## Installation

Setup everything:

```bash
yarn setup
```

## Development

Full stack:

```bash
yarn dev
```

Frontend:

```bash
yarn devFront
```

Backend:

```bash
yarn devBack
```

Start and stop mongo:

```bash
yarn startMongo
yarn stopMongo
```

Mongo CLI:

```bash
yarn devMongo
```
